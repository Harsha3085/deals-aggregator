About DealFinder

What is DealFinder?

DealFinder is an automated platform that finds and lists the best deals, coupons, and discounts from all major e-commerce websites every day.

How It Works

1. Automated Scraping

Our system automatically scans multiple e-commerce sites daily

2. Smart Filtering

We filter and rank deals based on quality and discount percentage

3. Easy Browsing

Users can browse, filter, and click through to purchase

Current Features

  • Daily automatic deal discovery
  • Multiple e-commerce site support (Amazon, Flipkart, etc.)
  • Smart categorization of deals
  • Filtering by category, price, and discount
  • Direct links to product pages
  • Admin panel for manual curation

For Developers

This is a demo project built with:

  • Backend: Django (Python)
  • Database: SQLite (development)
  • Frontend: Bootstrap 5
  • Scraping: BeautifulSoup, Requests
  • Hosting: Render.com (free tier)
  • Version Control: Git
